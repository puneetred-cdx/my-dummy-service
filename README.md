# My Dummy Service

This is a dummy service to demonstrate the routing patterns.

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

## Running the service

1. Compile the TypeScript code:
   ```bash
   npx tsc
   ```
2. Run the service:
   ```bash
   node build/app.js
   ```

The service will be running on `http://localhost:3001`.
